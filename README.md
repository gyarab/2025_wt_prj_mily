# RecipeBook

RecipeBook je webová aplikace vytvořená v Django, která slouží jako databáze receptů. 
Uživatelé si budou moci prohlížet jednotlivé recepty, jejich název, popis, seznam ingrediencí a postup přípravy.

Každý recept bude zařazen do určité kategorie (například dezerty, hlavní jídla nebo nápoje). 
Administrátor bude moci přidávat nové recepty, upravovat je nebo mazat přes administrační rozhraní.

Cílem projektu je vytvořit přehlednou webovou aplikaci, která pracuje s databází a umožňuje ukládání a zobrazování dat pomocí frameworku Django.
